# Backend_academia
Sistema de gerenciamento de academia desenvolvido em **Python** com **SQLite**, utilizando programação orientada a objetos e interface em terminal.

O projeto foi criado com o objetivo de praticar conceitos de desenvolvimento backend, banco de dados e organização de código.

---

## Sobre o projeto

O Backend Academia permite realizar o gerenciamento de alunos, treinos, exercícios e evolução física, simulando o funcionamento de um sistema utilizado por academias.

Todo o sistema é executado pelo terminal e utiliza SQLite para armazenamento dos dados.

---

## Tecnologias utilizadas

* Python 3
* SQLite3
* Rich (interface do terminal)
